## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Hazem-2002/Tailwind-Project.git
```

2. Navigate to the project folder:

```bash
cd Tailwind-Project
```

3. Open the project:

Open `index.html` in your browser
